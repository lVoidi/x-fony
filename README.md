# x-fony
### Genera numeros de donde quieras, cuando quieras


## Instalación y uso


### Clonamos el repositorio y entramos a la carpeta
```
git clone https://github.com/lVoidi/x-fony.git
cd x-fony
cd src
```

### Corremos el script
```
python index.py
```
